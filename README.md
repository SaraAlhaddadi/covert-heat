
# برنامج للتحويل من درجات حرارة مئوية الى درجات بالفهرنهايت
Prepared by : Shvm-k

## مثال 1
```
ادخل درجة الحرارة المئوية
20
20 دجة مئوية= 68.0 فهرناهايت(f°)
ادخل درجة الحرارة بالفهرناهايت
48
48 فهرناهايت= 8.88888888888889 درجة مئوية(c°)
```

## مثال 2
```
ادخل درجة الحرارة المئوية
l;
l; ليس رقم
ادخل درجة الحرارة بالفهرناهايت
5r
5r ليس رقم
```
